# RapportMeteoArduino

Le ficher tmpHumLum.ino est un code arduino pour récupérer les données des capteurs :
-Température
-Humidité
-Luminosité

Le ficher pollution.ino est un code arduino pour récupérer les données des capteurs :
-Pollution CO2
-Pollution TVOC
